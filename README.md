#  Python Advanced CLI Calculator & Tools

##  About The Project
A multi-functional command-line application written in Python. It goes beyond basic arithmetic by offering trigonometric functions, numeral system conversions, data plotting, and even a built-in terminal game. It serves as a great showcase of functional programming and control flow in Python.

##  Key Features
* **Basic & Advanced Math:** Perform standard calculations, power, and square roots.
* **Trigonometry:** Calculate `sin`, `cos`, `tan`, `cot`, `sec`, and `csc` with angle to radian conversion.
* **Number Base Conversions:** Easily convert between Decimal, Binary, and Octal formats.
* **Data Plotting:** Utilizes `matplotlib` to visualize data points directly from the script.
* **Mini-Game (Tic-Tac-Toe):** Take a break from math and play a classic 2-player XO game right in the terminal!

##  Built With
* **Python 3**
* **Math Module** (Built-in)
* **Matplotlib** (For data visualization)
